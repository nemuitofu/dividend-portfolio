# 高配当株ポートフォリオマネージャー 操作マニュアル

## 概要

SBI証券の保有証券CSVをアップロードするだけで、高配当株ポートフォリオを可視化・分析できるWebアプリです。
両学長スタイルの投資チェックリストに基づいて、ポートフォリオの健全性を評価できます。

---

## セットアップ

### 1. 依存パッケージのインストール

```bash
cd ~/sandbox_toromaru/dividend-portfolio
pip install -r requirements.txt
```

### 2. アプリの起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

---

## SBI証券からのCSVダウンロード方法

### 保有証券CSV（必須）

1. SBI証券にログイン
2. **口座管理** → **保有証券** を開く
3. 「ダウンロード」ボタンをクリック
4. `stockposition_YYYYMMDD.csv` という形式のファイルが保存される

### 資産残高CSV（任意）

1. **口座管理** → **資産残高** を開く
2. 「ダウンロード」ボタンをクリック
3. `assetbalance_YYYYMMDD.csv` という形式のファイルが保存される
   - アップロードすると、口座別（一般・NISA等）の内訳が表示されるようになる

## 基本的な使い方

### Step 1: CSVのアップロード

画面左のサイドバーに2つのアップロードエリアがあります。

| 項目 | ファイル名パターン | 必須 |
|------|------------------|------|
| 保有証券CSV | `stockposition_*.csv` | ◎ 必須 |
| 資産残高CSV | `assetbalance_*.csv` | △ 任意 |

アップロードが完了すると、自動的にデータが読み込まれ5つのタブが表示されます。

### Step 2: 配当データの取得

初回利用時は配当利回りデータがありません。

1. **「配当データ管理」タブ** を開く
2. **「yfinanceで一括取得」ボタン** をクリック
3. プログレスバーが表示され、全銘柄の配当利回りを自動取得
4. 取得したデータは `dividend_data.csv` にキャッシュされ、次回以降は再取得不要

> **注意:** yfinanceで取得できない銘柄は手動入力が必要です（後述）

## 各タブの説明

### タブ1: サマリー

ポートフォリオ全体の概況を確認できます。

#### KPIカード（上部）

| 指標 | 説明 |
|------|------|
| 総時価評価額 | 現在の全保有株の市場価値合計 |
| 評価損益 | 含み益・含み損の合計と損益率 |
| 年間配当金（予想） | 全銘柄の年間配当金合計 |
| ポートフォリオ利回り | 配当金 ÷ 時価評価額 |

#### 口座別内訳

資産残高CSVをアップロードした場合、口座種別（一般・NISA等）ごとの内訳がドーナツチャートで表示されます。

---

### タブ2: 銘柄一覧

保有全銘柄の詳細一覧テーブルです。

| 列名 | 説明 |
|------|------|
| 証券コード | 4桁の銘柄コード |
| 銘柄名 | 会社名 |
| 口座 | 保有口座種別 |
| セクター | 業種分類 |
| 現在値 | 現在の株価（円） |
| 平均取得単価 | 購入時の平均単価 |
| 保有株数 | 保有している株数 |
| 時価評価額 | 現在の評価額 |
| 評価損益 | 含み益・損 |
| 配当利回り | 色分け表示（下記参照） |
| 年間配当金 | この銘柄からの年間配当予想 |

#### 配当利回りの色分け

- **赤色**: 3%未満（目標未達）
- **緑色**: 3〜5%（理想的な範囲）
- **青色**: 5%超（高利回り）

---

### タブ3: セクター分散

両学長チェックリストとセクター配分を確認できます。

#### 両学長チェックリスト

| チェック項目 | 基準 |
|------------|------|
| ポートフォリオ利回り | 3〜5% |
| 保有銘柄数 | 20銘柄以上 |
| セクター数 | 10セクター以上 |
| 単一セクター集中 | 1セクター30%未満 |
| 配当データ取得率 | 80%以上 |

各項目に「✅ OK」または「⚠️ 要確認」が表示されます。

#### セクター配分チャート

ドーナツチャートで全セクターの保有比率を視覚的に確認できます。
テーブルには各セクターの時価評価額と割合も表示されます。

---

### タブ4: 配当データ管理

配当利回りデータの管理を行います。

#### 一括取得

「yfinanceで一括取得」ボタンで全銘柄の配当利回りを自動取得します。

#### 手動編集

yfinanceで取得できなかった銘柄（日本の小型株など）は、テーブルを直接編集して入力できます。

1. テーブルの「配当利回り (%)」列をクリック
2. 数値を入力（例: `3.5` と入力すると 3.5%）
3. 変更は自動保存されます

#### キャッシュについて

- データは `dividend_data.csv` にローカル保存されます
- 次回起動時は自動的に読み込まれるため、再取得不要です
- 配当金が変更された場合は「yfinanceで一括取得」を再実行してください

---

### タブ5: 生データ

アップロードしたCSVの生データを確認できます。
データの読み込みエラーや想定外の値が出た際のデバッグに使用します。

## セクターマスタのカスタマイズ

`sector_master.csv` を編集することで、銘柄のセクター分類をカスタマイズできます。

### ファイル形式

```csv
ticker,sector
1343,REIT・ETF
8306,銀行
9432,情報・通信
```

### 編集方法

1. `sector_master.csv` をテキストエディタまたはExcelで開く
2. 証券コードとセクター名を追加・修正する
3. UTF-8形式で保存する
4. アプリを再起動するか、CSVを再アップロードすると反映される

> **注意:** `sector_master.csv` に登録されていない銘柄は、証券コードの範囲から自動推定されます。

## よくある質問

### Q: 配当利回りが「-」や空欄になっている

**A:** yfinanceでデータを取得できていない銘柄です。

1. 「配当データ管理」タブを開く
2. 該当銘柄の配当利回り欄を手動で入力する

### Q: セクターが「不明」になっている

**A:** `sector_master.csv` に登録されておらず、自動推定もできない銘柄です。

`sector_master.csv` に `証券コード,セクター名` の形式で追加してください。

### Q: CSVのアップロードでエラーが出る

**A:** 以下を確認してください。

- ファイル名が `stockposition_*.csv` または `assetbalance_*.csv` の形式か
- SBI証券の正規のエクスポートファイルか（他の証券会社は非対応）
- ファイルが破損していないか

### Q: 評価損益の計算がおかしい

**A:** 資産残高CSV（`assetbalance_*.csv`）もアップロードすると、より正確な数値になる場合があります。

## データの流れ

```
SBI証券 CSV
    ↓ アップロード（サイドバー）
データ読み込み・正規化
    ↓
セッション状態に保存
    ↓
配当データ（dividend_data.csv）と結合
    ↓
セクターマスタ（sector_master.csv）でセクター付与
    ↓
5タブで可視化・分析
```
